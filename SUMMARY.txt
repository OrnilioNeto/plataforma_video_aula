🛠️ COMO ADICIONAR VÍDEOS
════════════════════════════════════════════════════════════════════════════════

1. Coloque seu arquivo MP4 em: static/videos/meu_video.mp4
2. Faça login como admin (admin/admin123)
3. Vá em "Novo Vídeo"
4. Preencha:
   - Título: Nome do vídeo
   - Arquivo: meu_video.mp4
   - Duração: em segundos (ex: 3600 para 1 hora)
5. Clique em "Cadastrar Vídeo"

Para calcular duração:
  Windows: ffprobe -v error -select_streams v:0 -show_entries stream=duration -of default=noprint_wrappers=1:nokey=1:precision=0 video.mp4
  Linux: ffprobe -v error -select_streams v:0 -show_entries stream=duration -of default=noprint_wrappers=1:nokey=1:precision=0 video.mp4

🐛 SOLUÇÃO DE PROBLEMAS
════════════════════════════════════════════════════════════════════════════════

Problema: "Porta 5000 em uso"
Solução: Mude a porta em app.py (última linha) para 5001, 8000, etc.

Problema: "ModuleNotFoundError: bcrypt"
Solução: pip install bcrypt

Problema: "Vídeo não aparece"
Solução: Confirme arquivo em static/videos/ e nome no BD

Problema: "Progresso não salva"
Solução: Verifique console (F12), confirme database.db existe

Mais soluções: Ver INSTALL.md
